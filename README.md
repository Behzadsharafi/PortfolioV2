## :toolbox: Getting Started

### :running: Run Locally

Clone the project

```bash
  git clone git@github.com:Behzadsharafi/PortfolioV2.git
```

Go to the project directory

```bash
  cd PortfolioV2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
